# symptom-triage

Suite for symptom intake / triage conversations (healthcare).

This suite is meant to stress what matters for downstream decision-making:
- **red flags** (chest pain, shortness of breath, severe headache, confusion)
- **negations** ("no", "denies", "not", "never") that flip meaning
- **severity scales** (0–10), **durations**, **onset times**
- **vitals** (temperature, SpO2, BP) and medication/allergy names

Audio guidance:
- include both calm and anxious delivery
- include background noise variants (street, room tone, HVAC)
- include faster/slower speech variants

Populate `audio/` with one clip per manifest row.
