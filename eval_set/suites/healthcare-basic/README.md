# healthcare-basic

Starter suite for healthcare-style voice agent QA.

Focus areas:
- proper nouns / names (patients, clinicians, clinics)
- entities (dates, phone numbers, dosages, appointment times)
- negations and meaning flips ("no" vs "yes", "do not", "denies")

Populate `audio/` with TTS outputs or recorded samples that correspond to `manifest.jsonl`.
Keep the expected scripts stable once you start tracking regressions.
