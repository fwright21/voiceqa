Independent audit

Voice agents fabricate. VoiceQA is the witness.

The independent audit layer for your STT, TTS, and full-duplex pipeline — catches hallucinations at every stage, across any vendor.
Start here
  • Select Demo Test Suite in the Eval Suites tab — 5 cases, all failure types covered.
  • Larger suites (hallucination, prosody, symptom triage) are also available if you want more coverage.
Tips
  • Audio stays on your machine — nothing is uploaded.
  • Use Save Baseline + Compare to track regressions.
  • Use the per-case Jump buttons to listen to flagged regions.

Run an eval suite

Run a curated suite to audit your voice agent across multiple cases — catch hallucinations, track regressions, compare against a baseline.

If audio files are missing, cases will FAIL fast with a “Missing audio file” reason.
